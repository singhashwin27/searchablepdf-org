# SearchablePDF.org - Issue Tracker

This repository is the official public issue tracker for [SearchablePDF.org](https://searchablepdf.org).

## About SearchablePDF.org

SearchablePDF.org is an OCR service that converts scanned PDFs into searchable, text-selectable documents.

## How to Report Issues

1. Check [existing issues](../../issues) to avoid duplicates
2. Click **New Issue** to create a report
3. Provide a clear title and detailed description
4. Include steps to reproduce the problem if applicable

## What to Include in Your Report

- Description of the issue
- Steps to reproduce
- Expected vs actual behavior
- Browser and device information (if relevant)
- Screenshots (if helpful)
